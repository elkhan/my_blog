```js
// Reading property attributes
// Itâ€™s also possible to retrieve the property descriptor for a given property by using the ECMAScript 5 Object.getOwnPropertyDescriptor() method.
// This method accepts two arguments: the object on which the property resides and the name of the property whose descriptor should be retrieved.
// The return value is an object with properties for configurable, enumerable, get, and set for accessor properties or configurable, enumerable,
// writable, and value for data properties.

const descriptorYear = Object.getOwnPropertyDescriptor(anotherBook, '_year')
console.log(descriptorYear.value) // 2005
console.log(descriptorYear.configurable) // false
console.log(typeof descriptorYear.get) // undefined

const descriptor = Object.getOwnPropertyDescriptor(anotherBook, 'year')
console.log(descriptor.value) // undefined
console.log(descriptor.enumerable) // false
console.log(typeof descriptor.get) // function
```
